Fixtures

Configure the environment with the fixtures.

         
         CREATE OR REPLACE PROCEDURE TEST_MY_TEST()
          BEGIN
           DECLARE EXPECTED INTEGER;
           DECLARE ACTUAL INTEGER;
           CALL DB2UNIT.REGISTER_MESSAGE('My first test');
           ...
           SET EXPECTED = 2;
           SET ACTUAL = ...;
           CALL DB2UNIT.ASSERT_INT_EQUALS('Same value', EXPECTED, ACTUAL);
          END @

Description

This is a testing framework for IBM DB2 written in SQL PL language. It is xUnit framework, that takes the same design as the well-known jUnit. The framework could be considered as a jUnit porting for DB2 SQL-PL.
